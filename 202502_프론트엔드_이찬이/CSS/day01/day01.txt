CSS 적용방법
 1. 인라인
HTML 요소 속성으로 직접 주입하는 방법
 - 재사용이 어렵다. (선택자가 없다)
 - 가독성을 해친다.
 - 가장 우선 순위가 높다. 장단점

 ex) 
 <div style="color: red" </div>

 2. 내부
 - 재사용이 가능하다.
 - 한 파일안에 html과 css를 함께볼 수 있다.
 - 가독성을 해치지 않음
 - 한 파일 안에서 밖에 사용이 안됨, 파일이 너무 길어질 수 있음 

 <head>
  <style>
   css 선택자 {
    color: red;
   }
  </style>

 </head>
 3. 외부 적용방법<가장많이 사용>

 - 파일을 하나 새로 생성하고 각각의 html 파일에 적용한다.
 - 여러 html 파일에 적용이 가능
 - 한파일에서 작성되지 않아 파일이 길어지지 않음
 -변경이 되면 다른 웹 페이지에도 영향이 발생할 가능성이 생김(주의)
